En esta tarea le vamos a mostrar en la pantalla del computador un conjunto de imágenes por treinta segundos. Usted va a ver 20 imágenes en la pantalla, por ejemplo, como esta:

Después de los treinta segundos, la pantalla desaparecerá y aparecerá una serie de pantallas donde en cada una le mostraremos tres imágenes. Para cada uno de estos le pediremos que seleccione la imagen que estaba incluida en el conjunto inicial de 20 imágenes.
